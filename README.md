# 🧠 Mental Health in Tech — Workforce Analysis

## 📊 Live Dashboard
👉 [View on Tableau Public](https://public.tableau.com/app/profile/subhadeep.koner/viz/MentalHealthCheckProject/Dashboard1)

## 📌 Project Overview
Analyzed OSMI Mental Health in Tech Survey data (1,243 respondents) to uncover workplace mental health trends across the tech industry.

## 🛠️ Tools Used
- Microsoft Excel — Data Cleaning
- Tableau Public — Data Visualization

## 📈 Key Findings
- 52.5% employees sought mental health treatment
- 78% report mental health impacts on work
- Age group 25–35 showed highest stress levels
- Larger companies offer significantly better mental health support

## 📂 Dataset
- Source: OSMI Mental Health in Tech Survey
- Records: 1,243 respondents
- Link: [Kaggle Dataset](https://www.kaggle.com/datasets/osmi/mental-health-in-tech-survey)
